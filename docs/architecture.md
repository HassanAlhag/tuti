# Tuti Architecture

Tuti is a luxury marketplace with three portal experiences and one API:

- Customer storefront
- Seller dashboard
- Admin operations console
- Backend API

The long-term structure is a monorepo where each portal can grow independently while sharing common utilities, API clients, auth state, design tokens, and UI primitives.

See [Milestone 001](./milestones/001-monorepo-portal-migration.md) for the active migration plan.
